# Python Portfolio

This repository contains four Python projects focused on data analysis and machine learning. Each project demonstrates end-to-end problem formulation, data handling, model construction, and outcome evaluation using real datasets.

---

## Projects

### 1. Gold Price Analysis
**Objective:**  
Analyze historical gold price data to identify trends, patterns, and relationships with time.

**Core Concepts:**  
Data cleaning, exploratory data analysis, time-series visualization, statistical interpretation.

**Technologies:**  
Python, Pandas, NumPy, Matplotlib, Seaborn.

**Outcome:**  
Produced visual and statistical insights into gold price fluctuations over time, enabling interpretation of long-term trends and short-term volatility.

---

### 2. Car Price Valuation Model
**Objective:**  
Predict the market price of used cars based on vehicle attributes.

**Core Concepts:**  
Feature engineering, regression modeling, train-test splitting, model evaluation.

**Technologies:**  
Python, Pandas, NumPy, Scikit-learn, Matplotlib.

**Outcome:**  
Built a regression model capable of estimating car prices from input features such as mileage, year, and specifications, demonstrating applied predictive modeling.

---

### 3. Fake News Detection
**Objective:**  
Classify news articles as real or fake using natural language processing techniques.

**Core Concepts:**  
Text preprocessing, vectorization (TF-IDF), supervised classification, model evaluation.

**Technologies:**  
Python, Pandas, Scikit-learn, NLP utilities.

**Outcome:**  
Trained a machine learning classifier that distinguishes fake news from real news with measurable accuracy, demonstrating applied NLP and classification workflows.

---

### 4. Neural Network Implementation
**Objective:**  
Implement and train a neural network model to understand core deep learning mechanics.

**Core Concepts:**  
Neural network architecture, forward and backward propagation, activation functions, loss optimization.

**Technologies:**  
Python, NumPy, machine learning fundamentals.

**Outcome:**  
Constructed a functional neural network model, reinforcing conceptual understanding of how neural networks learn and generalize from data.
